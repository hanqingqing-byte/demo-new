# Demo Feedback

本地原型，按 `demo.pen` 里的 5 个页面状态做了一个可直接运行的反馈站点：

- 设计师上传多张图片并生成分享链接
- 分享页可在手机和 PC 打开，查看 Demo 图片并提交反馈
- 设计师后台可以查看全部反馈，并区分未读反馈
- 所有数据都先保存在浏览器 `localStorage`，方便本地联调

## 运行

直接在当前目录启动一个静态服务：

```bash
cd /Users/hanqingqing/Desktop/评估
python3 -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173
```

## 说明

- 首次打开会自动写入两条示例 Demo 和几条反馈，方便直接看效果
- 新建 Demo 时上传的图片会压缩后存到浏览器本地
- 如果想清空数据，打开浏览器开发者工具，删除这个站点的 `localStorage` 即可
